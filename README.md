## Sintaxe Portugol

 ### Entrada 
  ```
   leia variavel
  ```
 ### Saída 
  ```
   escreval 'Exemplo'
  ```
 ### Atribuição 
 ```
  variavel exemplo = 10
 ```
 ### Operadores aritméticos 
 ```
  // Adicao
   variavel exemplo = valor + valor2
   
  // Subtracao
   variavel exemplo = valor - valor2
  
  // multiplicacao
   variavel exemplo = valor * valor2
  
  // divisao
   variavel exemplo = valor / valor2
  
  // potenciação
   variavel exemplo = valor ** valor2
 ```
 ### Condicionais 
  ```
   se ( valor == 10 ) {
    escreval 'O$valor$e$10'
   }
   senaose ( valor < 10 ) {
    escreval 'valor$menor$que$10'
   }
   senao {
    escreval 'valor$nao$conhecido'
   }
  ```
 ### Laço de repetição 
  ```
   enquanto ( valor < 10 ) {
    escreval valor
   }
  ```
