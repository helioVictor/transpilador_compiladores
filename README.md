## Sintaxe Portugol

 ### Entrada 
  ```
   leia ( variavel )
  ```
 ### Saída 
  ```
   escreval ( 'Exemplo' )
  ```
 ### Atribuição 
 ```
  variavel exemplo = 10
 ```
 ### Operadores aritméticos 
 ```
  // Adicao
   valor + valor2
   
  // Subtracao
   valor - valor2
  
  // multiplicacao
   valor * valor2
  
  // divisao
   valor / valor2
 ```
 ### Condicionais 
  ```
   se ( valor == 10 ) {
    escreval ( 'O valor e 10' )
   }
   senao se ( valor < 10 ) {
    escreval ( 'valor menor que 10' )
   }
   senao {
    escreval ( 'valor nao conhecido ')
   }
  ```
 ### Laço de repetição 
  ```
   enquanto ( valor < 10 ) {
    escreval ( valor )
   }
  ```
